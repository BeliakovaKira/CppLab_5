## Задание 1. Вычислить произведение нечетных отрицательных компонент файла.
1. Создаю бинарный файл со случайными числами от -10 до 10
2. Открываю и читаю созданный файл
3. Для каждого числа проверяю отрицательность и нечетность
4. Если соответствует - добавляю в произведение и вывожу число
5. Вывожу итоговое произведение или сообщение "не найдено"

## Задание 3. Информация о багаже пассажира описывается массивом, где каждый элемент содержит название единицы багажа (чемодан, сумка, коробка и т.д.) и ее массу. Дан файл, содержащий сведения о багаже нескольких пассажиров. Найти число пассажиров, имеющих более двух единиц багажа и число пассажиров, количество единиц багажа которых превосходит среднее число единиц багажа.

2. Записываю данные в бинарный файл
3. Читаю данные из файла

6. Для каждого пассажира проверяю: количество > среднего? -> добавляю во второй счетчик
7. Вывожу оба счетчика
